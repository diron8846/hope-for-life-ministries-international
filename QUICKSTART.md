# Quick Start Guide

## Get Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```
The app will automatically open at `http://localhost:5173`

### 3. Edit and Save
- Edit any file in `src/` directory
- Changes will hot-reload automatically in your browser

## Available Commands

```bash
# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview

# Run linter
npm run lint
```

## File Locations

| Type | Location |
|------|----------|
| React Components | `src/pages/` and `src/components/` |
| Styling | Tailwind classes + `src/index.css` |
| Images | `src/assets/` |
| Config | `vite.config.js`, `tailwind.config.js` |
